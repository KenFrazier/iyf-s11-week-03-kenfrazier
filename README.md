## What I Learned

This week strengthened my command-line fluency and Git workflow skills. Key takeaways:

- Using `find` and `grep` to search and filter files efficiently from the terminal
- Writing shell scripts with heredocs to automate repetitive tasks, like generating HTML boilerplate
- Managing remote repositories: cloning, pulling, and pushing changes with GitHub
- Creating and switching between branches, and resolving a real merge conflict when two branches edited the same content
- Writing clear, conventional commit messages (`type: short present-tense description`)
- Configuring `.gitignore` to keep unnecessary files out of version control
